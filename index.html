Project Overview

This DST-funded project develops semi-supervised learning for ROI extraction from satellite imagery.

Combining UNet + EfficientNet-B3, scSE attention, and PatchGAN discriminator.

Achieves 85.47% IoU with only 187 labeled images (80% cost reduction).
