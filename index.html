Project Overview

This research project develops cutting-edge semi-supervised segmentation methods for Region of Interest (ROI) extraction from hyperspectral remote sensing imagery. Our approach combines:

UNet Architecture

EfficientNet-B3 encoder with scSE attention mechanisms for robust feature extraction

Boundary-Aware

Morphological edge detection head for precise boundary delineation

Adversarial Training

PatchGAN discriminator for high-quality pseudo-label validation

Semi-Supervised

Efficient learning from both labeled and unlabeled datasets

Key Contributions

  • Novel semi-supervised framework combining FixMatch-style pseudo-labeling with adversarial validation
  • Boundary-aware architecture with dual segmentation and edge detection heads
  • Dynamic confidence masking using PatchGAN discriminator outputs
  • Adaptive threshold scheduling (TAU) for pseudo-label quality control
  • Production-ready modular implementation with comprehensive documentation

Methodology

Technical Approach

Model Architecture: UNet with EfficientNet-B3 encoder, scSE attention in decoder, and dual output heads for segmentation and boundary detection.

Training Strategy: Supervised learning on labeled data, pseudo-labeling on unlabeled data with confidence masking via PatchGAN discriminator.

Loss Functions: Combined Dice-BCE loss for segmentation, BCE loss for boundary detection, adversarial loss for discriminator training.

Optimization: Mixed-precision training with dynamic threshold scheduling and early stopping based on validation IoU.
